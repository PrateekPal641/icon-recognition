Deployed HuggingFace link - https://huggingface.co/spaces/Revrse/icon-classification

Steps to launch the app
#
For(mac/linux)
-> python3 -m venv venv
-> source venv/bin/activate
-> pip install -r requirements.txt
-> streamlit run app.py
#
For(Windows)
-> python3 -m venv venv
-> venv/Scripts/activate
-> pip install -r requirements.txt
-> streamlit run app.py
